# 🚖 GoCabs - Backend Cab Booking System

GoCabs is a backend application built with Java and Spring Boot, simulating a ride-booking system where users can register, book rides, and manage trips.

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

## 🔧 Features
- User registration & management
- Cab & driver entity management
- Ride booking system
- REST API architecture
- Exception handling

## ⚙️ Setup
1. Clone the repo
2. Create `application.properties` in `src/main/resources/`
3. Refer to `application-example.properties` for config
4. Run the application

## 🎯 Purpose
Built to learn real-world backend development with Spring Boot.
